소개

LangChain은 거대 언어 모델(LLMs)에 기반으로 동작하는 애플리케이션을 개발하기 위한 프레임워크입니다.

LangChain은 LLM 애플리케이션 라이프사이클의 모든 단계를 간소화 시켜줍니다.

  • 개발: LangChain의 오픈 소스 빌딩 블록 및 컴포넌트를 사용하여 애플리케이션을 구축하세요. 그 다음 써드파티 통합을 이용해서 바로 시작해보세요.
  • 프로덕션: LangSmith를 사용하여 체인을 검사, 모니터링 및 평가하면서 지속적으로 최적화하고 배포할 수 있습니다.
  • 배포: LangServe로 모든 체인을 API화 하세요.

구체적으로, 프레임워크는 다음과 같은 오픈 소스 라이브러리로 구성되어 있습니다:

  • @langchain/core: 기본 추상화 및 LangChain Expression Language.
  • @langchain/community: 써드 파티 통합.
    • 파트너 패키지들 (예 @langchain/openai, @langchain/anthropic, 등.): 일부 통합은 **@langchain/core**에만 의존하는 자체 경량 패키지로 더 분리되었습니다.
  • langchain: 애플리케이션의 인지 아키텍처를 구성하는 체인, 에이전트 및 검색 전략.
  • langgraph: 그래프에서 단계를 엣지와 노드로 모델링하여 LLM(대규모 언어 모델)을 사용해 강력하고 상태를 유지하는 다중 액터 애플리케이션을 구축합니다.
  • LangSmith: LLM 애플리케이션을 디버그, 테스트, 평가 및 모니터링할 수 있는 개발자 플랫폼.
note

이 문서는 자바스크립트 LangChain 라이브러리에 중점을 두고 있습니다. 파이썬 LangChain 라이브러리에 대한 문서는 여기를 참조하세요.

사용 가이드​

여기에서는 "어떻게 해야 하나요?" 유형의 질문에 대한 짧은 답변을 찾을 수 있습니다. 여기서는 주제를 깊이 다루지 않습니다. 이러한 자료는 튜토리얼과 API 참조에서 찾을 수 있습니다. 하지만, 이 가이드는 간단한 작업을 빠르게 해보는데 도움이 될 것입니다.

개념 가이드​

당신이 알아야 하는 LangChain의 모든 핵심 부분에 대한 소개글입니다! 여기에서는 모든 LangChain 개념에 대한 자세한 설명을 찾을 수 있습니다.

API 참조​

LangChain 파이썬 패키지의 모든 클래스와 메서드에 대한 전체 문서를 보려면 참조 섹션으로 이동하세요.

생태계​

🦜🛠️ LangSmith​

프로토타입에서 프로덕션으로 전환할 수 있도록 언어 모델 애플리케이션과 지능형 에이전트를 추적하고 평가합니다.

🦜🕸️ LangGraph​

LangChain 프리미티브 위에 구축된 LLM을 사용하여 상태를 유지하는 다중 액터 애플리케이션을 구축합니다.

추가적 리소스들​

보안​

LangChain을 사용하여 안전하게 개발하기 위해 보안 모범 사례를 읽어보세요.

통합​

LangChain은 우리 프레임워크와 통합되고 그 위에 구축되는 풍부한 도구 생태계의 일부입니다. 증가하고 있는 통합 목록을 확인해 보세요.

기여​

개발자 가이드에서 기여에 대한 가이드라인과 개발자 환경 설정에 대한 도움말을 확인하세요.
