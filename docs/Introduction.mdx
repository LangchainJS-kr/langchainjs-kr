# 소개

**LangChain**은 거대 언어 모델(LLMs)에 기반으로 동작하는 애플리케이션을 개발하기 위한 프레임워크입니다.

LangChain은 LLM 애플리케이션 라이프사이클의 모든 단계를 간소화 시켜줍니다.

- **개발**: **LangChain**의 **오픈 소스** [빌딩 블록](/docs/how_to/#langchain-expression-language-lcel) 및 [컴포넌트](/docs/how_to/)를 사용하여 애플리케이션을 구축하세요. 그 다음 [써드파티 통합](/docs/integrations/platforms/)을 이용해서 바로 시작해보세요.
- **프로덕션**: [LangSmith](/docs/langsmith/)를 사용하여 체인을 검사, 모니터링 및 평가하면서 지속적으로 최적화하고 배포할 수 있습니다.
- **배포**: [LangServe](https://www.langchain.com/langserve)로 모든 체인을 API화 하세요.

![LangChain Diagram](/img/langchain_stack_feb_2024.webp)

구체적으로, 프레임워크는 다음과 같은 오픈 소스 라이브러리로 구성되어 있습니다:

- **`@langchain/core`**: 기본 추상화 및 LangChain Expression Language.
- **`@langchain/community`**: 써드 파티 통합.
  - 파트너 패키지들 (예 **`@langchain/openai`**, **`@langchain/anthropic`**, 등.): 일부 통합은 **`@langchain/core`**에만 의존하는 자체 경량 패키지로 더 분리되었습니다.
- **`langchain`**: 애플리케이션의 인지 아키텍처를 구성하는 체인, 에이전트 및 검색 전략.
- **[langgraph](https://www.langchain.com/langserveh)**: 그래프에서 단계를 엣지와 노드로 모델링하여 LLM(대규모 언어 모델)을 사용해 강력하고 상태를 유지하는 다중 액터 애플리케이션을 구축합니다.
- **[LangSmith](/docs/langsmith)**: LLM 애플리케이션을 디버그, 테스트, 평가 및 모니터링할 수 있는 개발자 플랫폼.

:::note

이 문서는 자바스크립트 LangChain 라이브러리에 중점을 두고 있습니다. 파이썬 LangChain 라이브러리에 대한 문서는 [여기](https://python.langchain.com)를 참조하세요.

:::

## 생태계

### [🦜🛠️ LangSmith](/docs/langsmith)

프로토타입에서 프로덕션으로 전환할 수 있도록 언어 모델 애플리케이션과 지능형 에이전트를 추적하고 평가합니다.

### [🦜🕸️ LangGraph](/docs/langgraph)

LangChain 프리미티브 위에 구축된 LLM을 사용하여 상태를 유지하는 다중 액터 애플리케이션을 구축합니다.
